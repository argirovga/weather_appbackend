Available requests
Weather and clothes mechanisms
Получить данные о погоде сейчас
api/weather/lat="широта"&lon="долгота"
See example(Moscow)
Получить прогноз погоды
api/weather/forecast/lat="широта"&lon="долгота"
See example(Moscow)
Получить все возможные данные
api/all_in_one/lat="широта"&lon="долгота"&user_id="id юсера"
See example(Moscow)
Получить данные по одежде сейчас
api/clothes/lat="широта"&lon="долгота"&user_id="id юсера"
See example(Moscow)
